# Buku UMKM - UMKM Accounting & Tax Helper

Buku UMKM adalah aplikasi web sederhana yang dirancang untuk membantu pelaku UMKM mengelola pembukuan dan perhitungan pajak dengan lebih mudah. Aplikasi ini memungkinkan pengguna untuk mencatat pemasukan, mengelola pengeluaran, membuat invoice, serta memperkirakan kewajiban pajak dalam satu tempat.

Dibangun khusus untuk UMKM (Usaha Mikro, Kecil, dan Menengah), aplikasi ini berfokus pada kesederhanaan dan kemudahan penggunaan sehingga pemilik usaha dapat mengelola keuangan tanpa perlu menggunakan software akuntansi yang rumit.

## Fitur

- Pencatatan pemasukan dan pengeluaran
- Pembukuan sederhana untuk UMKM
- Manajemen invoice
- Perhitungan pajak dasar
- Ringan dan mudah digunakan

## Tujuan

Proyek ini bertujuan untuk mempermudah pengelolaan keuangan bagi UMKM dengan menyediakan alat bantu akuntansi dan pajak yang praktis, sederhana, dan mudah diakses.
